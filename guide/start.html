安装pnpm ​

组件库使用 pnpm 作为包管理工具，请确保已安装 pnpm。

sh
npm config set registry https://registry.npmjs.org
npm install -g pnpm

项目目录 ​

业务组将库采用 pnpm 的 monorepo 多包管理模式。 项目主要目录介绍

  • demo包 - 业务组件开发验证包
  • docs包 - 业务组件开发文档包
  • packages包 - 业务组件包

packages包中主要包含

  • bui包 - 用于业务组件的开发
  • platform包 - 用于业务方法层面的开发
  • types包 - 用于业务组件的类型定义

注意：

业务组件的开发主要在bui包中进行。
